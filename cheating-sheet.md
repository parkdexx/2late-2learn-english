## 2026-04-07

### 1. 조동사 뒤에는 동사원형이 온다
- 조동사 : can, could, may, might, will, would, shall, should, must, have to, need to, ought to

```
He will be a great leader

//am, are, is 는 주어에 맞춘 형태이고 원형은 be 임
```

### 2. 부사가 수식하는 것
- 부사는 동사, 형용사, 부사, 문장 전체 를 수식한다
- 부사는 명사를 수식하지 않는다

### 3. 시점을 나타내는 전치사 들 (더 많은데 오늘은 이거 봄)
- among: '~사이에(셋 이상)'
- between: '~사이에(둘 사이)'
- within: '~이내에'
- before: '~이전에'

### 4. 자동사 들 (더 많은데 우선 이거만이라도 제발)
- 존재/발생
  - exist (존재하다)
  - occur/happen (발생하다)
  - appear (나타나다)
- 이동
  - go/come (가다/오다)
  - arrive (도착하다)
  - depart (출발하다)
- 변화/상태
  - become/seem (되다/~인 것 같다)
  - remain (남다)
- 일/기능
  - work (일하다)
  - operate (작동하다)
  - function (기능하다)
- 증감
  - rise/fall (오르다/떨어지다)
  - increase/decrease (증가/감소하다)

## 2026-04-06

### 1. -ed 형태 동사, 형용사 구분
- -ed 형태는 동사일수도 있고, 형용사일수도 있음
- 영어 문장에서 진짜 동사(본동사) 는 딱 하나여야 함
  - 대충 주어 가까이 본동사 찾고 나서 -> 나머지는 다 형용사로 생각해버리자
- 해석의 차이로 구분
  - 동사의 경우 : ~했다
    ```
    He closed the door
    ``` 
  - 형용사의 경우 : ~된, ~한
    ```
    The closed door
    ```

### 2. 전치사 뒤에 일반명사? 가 와야해요 동명사? 가 와야해요?
- 일반명사, 동명사 둘 다 올 순 있음
- 일반명사가 오는 경우 : 전치사 + 일반명사
  ```
  after the conduction
  ```
- 동명사가 오는 경우 : 전치사 + 동명사 + 목적어
  ```
  after conducting the test
  ```

### 3. to 가 부정사로 쓰인건지? 전치사로 쓰인건지?
- to 부정사 일 경우, + 동사원형 이 와야 함
- to(전치사) 일 경우, + 일반명사, 동명사가 와야 함

- 아래 16개는 to(전치사) 라고 생각하자 (나머지는 다 to 부정사)

  - be committed to + 명사/동명사

  - object to + 명사/동명사
  - be dedicated to + 명사/동명사
  - be devoted to + 명사/동명사
  - when it comes to + 명사/동명사
  - be subject to + 명사/동명사
  - with a view to + 명사/동명사
  - react to + 명사/동명사
