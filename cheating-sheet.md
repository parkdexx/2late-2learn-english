## 2026-04-16
```
The shipment cannot be dispatched unless payment ______ in full.

A) receives
B) is received
C) received
D) has received
```

### 풀이
- A : receive 가 쓰이게 되면 능동태가 되어 해석이 이상해짐
- B : 수동태, 뒤에 in full 이라는 부사구와 어울림
- C : 과거형, 수동태가 되려면 be 동사가 필요함
- D : 현재완료 수동태, 문맥상 현재 시점의 상황이므로 현재형이 더 적절함 / unless 절은 미래 의미라도 현재시제로 표현함

### 해석
- The shipment cannot be dispatched: 그 배송은 발송될 수 없다 (주절)
- unless payment is received in full: 대금이 전액 지급되지 않는다면 (부사절)

### 단어
- shipment 📦	명사	배송, 선적
- dispatch 🚀	동사	발송하다, 급파하다
- payment 💰	명사	지불, 결제
- in full 💯	부사구	전액, 완전히

```
The director’s ______ to detail impressed the clients during the presentation.

A) attend
B) attentive
C) attention
D) attentively
```

### 풀이
- A : attend 는 동사, 빈칸에는 명사가 와야 함
- B : attentive 는 형용사, 소유격 뒤 명사가 와야 함
- C : 명사, 뒤에 to detail 이라는 명사와 어울림
- D : 부사, 명사를 꾸며줄 수 없음

### 해석
- The director’s attention to detail: 그 이사의 세심한 주의력 (주절)
- impressed the clients: 고객들에게 깊은 인상을 주었다 (동사구)
- during the presentation: 발표 중에 (시간 부사구)

### 단어
- director 👤	명사	이사, 감독
- attention 💡	명사	주의력, 관심
- detail 🔍	명사	세부 사항
- impressed ✨	동사	깊은 인상을 주다
- presentation 🎤	명사	발표

## 2026-04-15
```
The manager requested that all employees submit their weekly reports ------- the end of the business day on Friday.

(A) within
(B) by
(C) until
(D) for
```

### 풀이
- (A) within : ~이내에 / 뒤에 기간이 나와야 함 / Friday 라는 시점이 왔기 때문에 안됨
- (B) by : ~까지 / 뒤에 시점과, 완료의 상태가 나와야 함 / Friday 라는 시점과, submit 하면 완료되는 상황이라서 ok
- (C) until : ~까지 / 뒤에 시점과, 지속의 상태가 나와야 함 / Friday 시점은 ok 인데, submit 을 지속하는건 아니니까 ng
- (D) for : ~동안 / 뒤에 기간이 나와야 함 / Friday 라는 시점이 왔기 때문에 안됨

### 해석
- The manager requested (주어 + 동사): 매니저는 요청했다
- that all employees submit (접속사 + 목적어절): 모든 직원이 제출할 것을
- their weekly reports (제출의 목적어): 그들의 주간 보고서를
- by the end of the business day (시간 전치사구 1): 업무 시간 종료 전까지
- on Friday (시간 전치사구 2): 금요일에

## 2026-04-14

```
1. 다음 빈칸에 알맞은 단어를 고르세요.

The board of directors is currently searching for a candidate who possesses __________ leadership skills and extensive experience in the industry.

(A) except
(B) exception
(C) exceptional
(D) exceptionally
```

### 풀이

- A : 동사 prossesses 의 목적어 자리인데 전치사가 왜? 
- B : exception + leadership skills 는 복합명사가 아님
- C : 뒤에 올 명사 leadership skills 를 꾸며주는 형용사로 정답
- D : 뒤에 올 명사 leadership skills 를 꾸며줘야 하는데 부사가 왜?

### 해석

- The board of directors: 이사회 (주어)
- is currently searching for: 현재 ~을 찾고 있다 (동사구)
- a candidate: 후보자를 (목적어)
- who possesses: (~을) 갖춘 (관계대명사절 시작)
- exceptional leadership skills: 뛰어난 리더십 기술을 (관계절 내 목적어 1)
- and extensive experience: 그리고 폭넓은 경험을 (관계절 내 목적어 2)
- in the industry: 업계에서의 (수식어)

### 단어
- currently ⏱️	부사	현재, 지금
- candidate 👤	명사	후보자, 지원자
- possesses ✨	동사	(자질·특징 등을) 갖추다, 소유하다
- exceptional 🌟	형용사	뛰어난, 이례적인
- extensive 🌊	형용사	폭넓은, 광범위한

## 2026-04-07

### 1. 조동사 뒤에는 동사원형이 온다
- 조동사 : can, could, may, might, will, would, shall, should, must, have to, need to, ought to

```
He will be a great leader

//am, are, is 는 주어에 맞춘 형태이고 원형은 be 임
```

### 2. 부사가 수식하는 것
- 부사는 동사, 형용사, 부사, 문장 전체 를 수식한다
- 부사는 명사를 수식하지 않는다

### 3. 시점을 나타내는 전치사 들 (더 많은데 오늘은 이거 봄)
- among: '~사이에(셋 이상)'
- between: '~사이에(둘 사이)'
- within: '~이내에'
- before: '~이전에'

### 4. 자동사 들 (더 많은데 우선 이거만이라도 제발)
- 존재/발생
  - exist (존재하다)
  - occur/happen (발생하다)
  - appear (나타나다)
- 이동
  - go/come (가다/오다)
  - arrive (도착하다)
  - depart (출발하다)
- 변화/상태
  - become/seem (되다/~인 것 같다)
  - remain (남다)
- 일/기능
  - work (일하다)
  - operate (작동하다)
  - function (기능하다)
- 증감
  - rise/fall (오르다/떨어지다)
  - increase/decrease (증가/감소하다)

## 2026-04-06

### 1. pp 형태 동사, 형용사 구분
- pp 형태는 동사일수도 있고, 형용사일수도 있음
- 영어 문장에서 진짜 동사(본동사) 는 딱 하나여야 함
  - 대충 주어 가까이 본동사 찾고 나서 -> 나머지는 다 형용사로 생각해버리자
- 해석의 차이로 구분
  - 동사의 경우 : ~했다
    ```
    He closed the door
    ``` 
  - 형용사의 경우 : ~된, ~한
    ```
    The closed door
    ```

### 2. 전치사 뒤에 일반명사? 가 와야해요 동명사? 가 와야해요?
- 일반명사, 동명사 둘 다 올 순 있음
- 일반명사가 오는 경우 : 전치사 + 일반명사
  ```
  after the conduction
  ```
- 동명사가 오는 경우 : 전치사 + 동명사 + 목적어
  ```
  after conducting the test
  ```

### 3. to 가 부정사로 쓰인건지? 전치사로 쓰인건지?
- to 부정사 일 경우, + 동사원형 이 와야 함
- to(전치사) 일 경우, + 일반명사, 동명사가 와야 함

- 아래 16개는 to(전치사) 라고 생각하자 (나머지는 다 to 부정사)

  - be committed to + 명사/동명사
  - prior to + 명사/동명사
  - object to + 명사/동명사
  - be dedicated to + 명사/동명사
  - be devoted to + 명사/동명사
  - when it comes to + 명사/동명사
  - be subject to + 명사/동명사
  - with a view to + 명사/동명사
  - react to + 명사/동명사
  - respond to + 명사/동명사
  - in addition to + 명사/동명사
  - be accustomed to + 명사/동명사
  - be used to + 명사/동명사
  - contribute to + 명사/동명사
  - be opposed to + 명사/동명사
