## cheating-sheet.md

문법 개념을 날짜별로 기록한 노트입니다.
잘못된 내용이 있을 수 있으니 참고용으로만 사용하세요.

주요 내용:
- 수동태 (`be + pp`)
- 조동사 뒤 동사원형
- 부사의 역할과 수식 대상
- 전치사 정리 (`among`, `between`, `within` 등)
- `to` 부정사 vs 전치사 구분 (16개 패턴)
- `-ed` 형태의 동사/형용사 구분
- 자동사 목록

→ [cheating-sheet.md 바로가기](./cheating-sheet.md)

## voca.md

영어 예문을 슬래시(`/`)로 끊어 읽으며 각 단어의 뜻, 품사, 문장 내 역할을 분석한 어휘 노트입니다.

예시 문장:
> *The newly appointed executive / diligently monitored / the compliance / of all departments / with the revised / environmental safety regulations.*
